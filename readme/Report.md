## Тема
Разработка мессенджера для студентов и преподавателей кафедры

## Цель работы
Создать веб-сервис для коммуникации внутри кафедры, который объединяет студентов и преподавателей в единой системе. Основной фокус — обеспечить базовую платформу для регистрации, обмена сообщениями и управления профилем, начиная с полного отсутствия готового функционала. Важно спроектировать удобный интерфейс и надёжную серверную часть так, чтобы проект можно было развивать дальше как учебный инструмент.

## Задача
Разработать прототип мессенджера для учебного взаимодействия, включающий:
- регистрацию и вход пользователей;
- личный кабинет с профилем и аватаром;
- возможность вести короткие публикации или заметки на «стене»;
- создание списка контактов/друзей;
- личные чаты в реальном времени;
- базовую систему авторизации и защиты доступа.

Дополнительно задача предполагает исследование архитектуры, выделение ключевых компонентов и создание основы, на которой можно строить дальнейшее развитие продукта: обмен медиа, групповые беседы, уведомления и мобильный интерфейс.
